# API Reference

Welcome to the **Pt.Okx.Sdk** API reference.

Here you will find detailed documentation on all the namespaces, classes, interfaces, and methods available in the SDK.

Please use the table of contents on the left (or the navigation menu) to browse through the API documentation.

## Key Namespaces

- **`Pt.Okx.Sdk.Clients`**: Contains client interfaces for interacting with the OKX exchange (e.g., `IOkxClient`, `ITradeClient`).
- **`Pt.Okx.Sdk.Strategy`**: Interfaces and base classes for developing trading strategies (e.g., `IStrategy`, `StrategyBase`).
- **`Pt.Okx.Sdk.Indicators`**: Base classes and services for custom indicators.
- **`Pt.Okx.Sdk.Common`**: Common utilities and the `ApiResult` pattern for standardized error handling.
